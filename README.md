# 🚨 Campus Guardian – AI Powered Campus Safety System

Campus Guardian is a real-time, AI-powered campus safety solution that allows students to send emergency SOS alerts with automatic GPS location detection and intelligent response support.

--- 

## 🧠 Problem Statement
Campus harassment and emergency situations require **instant response**, but:
- Victims may not be able to speak or explain location
- Delayed help increases risk
- Manual reporting systems are slow

---

## 💡 Solution
Campus Guardian provides:
- One-tap SOS alert
- Automatic live location detection
- AI-based danger analysis
- Real-time admin monitoring
- Nearest security guard identification

---

## 🚀 Key Features
- 🚨 One-Tap Emergency SOS
- 📍 Automatic GPS Location Detection
- 🧠 AI Danger Analysis (Risk Level & Message)
- 🛡️ Admin Dashboard (Live Alerts)
- 🗺️ Real-Time Map Visualization
- 🚔 Nearest Security Guard Detection
- 🔥 Firebase Realtime Database Integration

---

## 🏗️ System Architecture
User → Streamlit App → GPS + AI Analysis → Firebase → Admin Dashboard → Nearest Guard

---

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **Backend:** Firebase Realtime Database
- **AI:** OpenAI API (with fallback)
- **Mapping:** Streamlit Map (Google Maps based)
- **Language:** Python
