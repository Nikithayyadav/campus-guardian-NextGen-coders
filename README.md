# campus-guardian-NextGen-coders
AI-powered campus safety and emergency system using Google Maps, Firebase and AI to provide real-time SOS alerts, live location tracking, and risk zone monitoring. 

It uses Google Maps, Firebase, and AI to provide:
- One-tap SOS alerts
- Live location tracking
- Risk zone monitoring
- Real-time admin dashboard

The goal is to help students get instant help during emergencies and allow campus authorities to respond faster and smarter.

## Tech Stack
- Python
- Google Maps API
- Firebase
- Google Gemini / AI
- Web Frontend
